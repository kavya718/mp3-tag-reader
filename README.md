# MP3 Tag Reader

An MP3 Tag Reader project developed in C to read, display, and edit MP3 metadata information.

## Features
- Read ID3 tags
- Display song details
- Edit MP3 metadata
- User-friendly command-line interface

## Technologies Used
- C Programming
- File Handling
- Structures

## Applications
- Audio metadata management
- MP3 file organization
