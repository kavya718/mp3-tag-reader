# 🎵 MP3 Tag Reader

A command-line application built in **C** that reads and edits **ID3v2 metadata tags** of MP3 audio files. It allows users to view and modify information such as the title, artist, album, genre, and year without affecting the audio content.

## ✨ Features

🎼 **Read MP3 Metadata** – Displays title, artist, album, genre, year, and other ID3 tags

✏️ **Edit Metadata Tags** – Modify selected fields without changing the audio data

📁 **Binary File Processing** – Reads and writes MP3 files at the binary level

🔍 **ID3 Tag Validation** – Verifies the presence of valid ID3 metadata

💾 **Save Updated Tags** – Writes modified metadata back to the MP3 file

⚡ **Command-Line Interface** – Easy-to-use interface for reading and editing tags

🧩 **Modular Design** – Separate modules for reading, editing, validation, and file handling
